这份编辑器完整功能梳理，分模块讲清楚
 
一、登录权限模块
 
1. 管理员密码验证，默认密码  admin123 
2. 记住密码本地缓存，下次打开自动登录，不用重复输密码
 
二、页面跳转导航
 
1. 跳转博客主页、图文相册（新开标签页）
2. 返回笔记管理列表 admin-list.html、网站首页 index.html
 
三、图片素材面板（折叠展开）
 
1. 批量选中本地图片，生成 Termux  cp  复制命令（适配截图/相机/微信图片路径）
2. 一键复制完整  data.js  源码，直接粘贴 nano 编辑
3. 快速插入代码块模板
4. 多选图片记录本地路径，后续一键复制终端上传脚本
 
四、备份/导出/下载功能
 
1. 下载无时间戳  data.js 
2. 下载带时间戳  data_2026-06-26-xxx.js ，留存历史版本
3. 完整导出笔记备份  notes.json （大容量全数据）
4. 导入本地 json/js 备份文件，恢复全部笔记
5. Ctrl+S 快捷键一键复制 data.js 代码
 
五、笔记基础编辑项
 
1. 标题输入框、自定义分类标签
2. 双编辑模式切换：富文本可视化编辑 / Markdown纯文本
3. Markdown 实时预览面板，切换模式自动互转 HTML ↔ MD
4. 撤销 / 重做（富文本原生、MD单独栈记录）
 
六、底部工具栏（核心格式化功能）
 
基础文字样式（常用区）
 
1. 字号选择 12~24px
2. 文字颜色、文字背景高亮取色器
3. 加粗 B、斜体 I、下划线 U、删除线 S̶
4. 本地单图上传、批量粘贴网络图床链接
5. 代码块、表格、短分割线
 
折叠更多工具面板（⋮展开）
 
1. 排版：居中、H1/H2/H3 三级标题、行内代码
2. 列表：无序列表、有序列表、任务复选框、空白/已勾选方框
3. 引用：普通引用 >、嵌套引用 >>
4. 链接插入、普通图片弹窗、固定宽度图片
5. 公式：行内公式、块级 Latex 公式
6. 复制选中文字、段落上下移动按钮
7. 长分割线、取色器、一键切换 MD/富文本模式
 
七、图片插入全套逻辑（适配Termux手机路径）
 
1. 单张本地图片上传，自动区分目录：
- 相机照片： ~/storage/dcim/Camera 
- 截图： ~/storage/Pictures/Screenshots 
- 微信图片： ~/storage/Download 
2. 网络图床图片弹窗，支持自定义图片描述
3. 带宽度限制图片插入  ![图](url){width=xxx} 
4. 批量多行图床链接一键批量插入
5. 自动重命名本地图片为  IMG_时间戳.后缀 ，避免中文乱码
6. 生成完整 Git+cp 上传终端脚本，复制到 Termux 一键上传图片到仓库
 
八、表格双向转换
 
- MD 表格 ↔ 富文本 HTML 表格，切换模式不会丢失表格结构
- 自带边框样式，手机端清晰可见
 
九、代码块、引用、列表智能换行
 
Markdown 模式下回车自动续上格式：
 
- 加粗/斜体/删除线/引用/无序列表/任务列表 回车自动补全前缀
- 代码块自动把光标定位到两行 ``` 中间
 
十、本地持久化存储
 
1. 编辑过程临时缓存笔记到 localStorage，刷新不丢内容
2. 新增/修改笔记存入全局  window.NOTE_DATA  数组
3. 每条笔记自动生成唯一ID、记录修改时间、标题、分类、富文本内容
 
十一、浏览器兼容适配
 
1. 适配 Via / Vivo / QQ 浏览器，修复文件上传真实文件名读取问题
2. 软键盘弹出自动适配页面底部边距，不遮挡编辑器
3. 移动端禁止双指缩放、适配手机安全底部小黑条
4. 弹窗遮罩、点击空白关闭弹窗
 
十二、Git 仓库配套辅助功能
 
1. 一键生成完整上传命令：创建images文件夹 + cp复制图片 + git提交推送
2. 导出完整笔记库，直接同步到 github.com:sxgpyjg/notes.git
3. 分版本下载 data.js，方便回退历史提交
 
十三、小工具细节
 
1. 鼠标悬浮按钮显示功能提示（data-tip）
2. 当前激活格式按钮高亮（加粗/标题/居中会变色标记）
3. 复制选中文字功能，兼容无剪贴板权限老旧浏览器
4. 所有弹窗、输入框自适应手机宽度
 
补充区分
 
1. 静态页面，纯前端，不需要后端，所有数据存在浏览器本地、导出文件后手动上传仓库；
2. 完全适配你之前的 Termux + GitHub 笔记仓库工作流；
3. 富文本和 Markdown 双向无损转换，专门适配图文笔记、开发文档写作。
